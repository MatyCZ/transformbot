Transformation Actions

Powerful operations for manipulating and formatting data in TransformBot

Overview String Actions Number Actions Date Actions Normalization Actions Custom Actions

Overview

Transformation actions are functions that modify, format, or normalize data without changing its base type. They are designed to be composable and can be chained together using the pipe() function.

Actions are organized into several categories based on the type of data they operate on and the transformations they perform:

  • String Actions - For text manipulation, formatting, and cleaning
  • Number Actions - For numeric formatting and range constraints
  • Date Actions - For date formatting and manipulation
  • Normalization Actions - For handling edge cases like null values
  • Custom Actions - For creating your own transformations

String Actions

These actions operate on string values, providing various text transformations and formatting operations.

Text Case Transformations

toUpperCase()

Converts a string to uppercase.

const result = tb.transform(tb.pipe(tb.string(), tb.toUpperCase()), "hello");
// result: "HELLO"
toLowerCase()

Converts a string to lowercase.

const result = tb.transform(tb.pipe(tb.string(), tb.toLowerCase()), "HELLO");
// result: "hello"
toCapitalCase()

Converts a string so that each word starts with uppercase.

const result = tb.transform(tb.pipe(tb.string(), tb.toCapitalCase()), "hello world");
// result: "Hello World"
toCamelCase()

Converts a string to camelCase format.

const result = tb.transform(tb.pipe(tb.string(), tb.toCamelCase()), "hello world");
// result: "helloWorld"
toPascalCase()

Converts a string to PascalCase format.

const result = tb.transform(tb.pipe(tb.string(), tb.toPascalCase()), "hello world");
// result: "HelloWorld"
toSnakeCase()

Converts a string to snake_case format.

const result = tb.transform(tb.pipe(tb.string(), tb.toSnakeCase()), "hello world");
// result: "hello_world"
toKebabCase()

Converts a string to kebab-case format.

const result = tb.transform(tb.pipe(tb.string(), tb.toKebabCase()), "hello world");
// result: "hello-world"
toDotCase()

Converts a string to dot.case format.

const result = tb.transform(tb.pipe(tb.string(), tb.toDotCase()), "hello world");
// result: "hello.world"

Whitespace Handling

trim()

Removes whitespace from both ends of a string.

const result = tb.transform(tb.pipe(tb.string(), tb.trim()), "  hello  ");
// result: "hello"
trimStart()

Removes whitespace from the beginning of a string.

const result = tb.transform(tb.pipe(tb.string(), tb.trimStart()), "  hello  ");
// result: "hello  "
trimEnd()

Removes whitespace from the end of a string.

const result = tb.transform(tb.pipe(tb.string(), tb.trimEnd()), "  hello  ");
// result: "  hello"
stripMultiSpaces()

Removes multiple consecutive spaces, replacing them with a single space.

const result = tb.transform(tb.pipe(tb.string(), tb.stripMultiSpaces()), "hello    world");
// result: "hello world"
stripSpaces()

Removes all spaces from a string.

const result = tb.transform(tb.pipe(tb.string(), tb.stripSpaces()), "hello world");
// result: "helloworld"
stripNewlines()

Removes all newline characters from a string.

const result = tb.transform(tb.pipe(tb.string(), tb.stripNewlines()), "hello\nworld");
// result: "helloworld"

Character Filtering

toAlpha()

Removes all non-alphabetic characters.

const result = tb.transform(tb.pipe(tb.string(), tb.toAlpha()), "hello123");
// result: "hello"
toAlphanumeric()

Removes all non-alphanumeric characters.

const result = tb.transform(tb.pipe(tb.string(), tb.toAlphanumeric()), "hello-123!");
// result: "hello123"
toDigits()

Extracts only digits from a string.

const result = tb.transform(tb.pipe(tb.string(), tb.toDigits()), "abc123def456");
// result: "123456"
stripTags()

Removes HTML/XML tags from a string.

const result = tb.transform(tb.pipe(tb.string(), tb.stripTags()), "<p>hello</p>");
// result: "hello"
stripDiacritics()

Removes diacritical marks from a string.

const result = tb.transform(tb.pipe(tb.string(), tb.stripDiacritics()), "résumé");
// result: "resume"

String Manipulation

padStart(length, padString)

Pads the start of a string with a specified character until it reaches the desired length.

const result = tb.transform(tb.pipe(tb.string(), tb.padStart(5, "0")), "42");
// result: "00042"
padEnd(length, padString)

Pads the end of a string with a specified character until it reaches the desired length.

const result = tb.transform(tb.pipe(tb.string(), tb.padEnd(5, "0")), "42");
// result: "42000"
repeat(count)

Repeats a string a specified number of times.

const result = tb.transform(tb.pipe(tb.string(), tb.repeat(3)), "abc");
// result: "abcabcabc"
replace(searchValue, replaceValue)

Replaces occurrences of a substring with another string.

const result = tb.transform(tb.pipe(tb.string(), tb.replace("world", "universe")), "hello world");
// result: "hello universe"
slice(start, end)

Extracts a section of a string and returns it as a new string.

const result = tb.transform(tb.pipe(tb.string(), tb.slice(0, 5)), "hello world");
// result: "hello"
substring(start, end)

Returns the part of a string between two indices.

const result = tb.transform(tb.pipe(tb.string(), tb.substring(6, 11)), "hello world");
// result: "world"

Encoding/Decoding

encodeURI()

Encodes a string as a URI component.

const result = tb.transform(tb.pipe(tb.string(), tb.encodeURI()), "hello world");
// result: "hello%20world"
decodeURI()

Decodes a URI-encoded string.

const result = tb.transform(tb.pipe(tb.string(), tb.decodeURI()), "hello%20world");
// result: "hello world"
normalize(form)

Normalizes a Unicode string.

const result = tb.transform(tb.pipe(tb.string(), tb.normalize("NFC")), "\u0041\u030A");
// result: "\u00C5" (Å)
toWellFormed()

Replaces all lone surrogates in a string with the Unicode replacement character U+FFFD.

const result = tb.transform(tb.pipe(tb.string(), tb.toWellFormed()), problematicString);
// result: String with surrogates replaced

Combined String Actions Example

This example demonstrates how to combine multiple string actions to create a username formatter:

import * as tb from 'transformbot';

// Create a reusable transformation for usernames
const usernameFormatter = tb.pipe(
  tb.string(),        // Ensure we're working with a string
  tb.trim(),          // Remove surrounding whitespace
  tb.stripTags(),     // Remove any HTML tags
  tb.toAlphanumeric(),// Keep only alphanumeric characters
  tb.toLowerCase(),   // Convert to lowercase
  tb.toNull()         // Convert empty strings to null
);

// Example usage:
const dirtyUsername = " <script>alert('hi')</script>John Doe123! ";
const cleanUsername = tb.transform(usernameFormatter, dirtyUsername);
// Result: "johndoe123"

Number Actions

These actions operate on numeric values, providing formatting and constraint options.

Number Type Conversions

toInteger()

Converts a number to an integer by removing the decimal part.

const result = tb.transform(tb.pipe(tb.number(), tb.toInteger()), 42.75);
// result: 42
toFloat()

Ensures a number is treated as a floating-point number.

const result = tb.transform(tb.pipe(tb.number(), tb.toFloat()), 42);
// result: 42.0

Number Formatting

toFixed(digits)

Formats a number using fixed-point notation with a specified number of digits.

const result = tb.transform(tb.pipe(tb.number(), tb.toFixed(2)), 42.125);
// result: "42.13"
toPrecision(precision)

Returns a string representing a number to a specified precision.

const result = tb.transform(tb.pipe(tb.number(), tb.toPrecision(4)), 42.125);
// result: "42.13"
toExponential(fractionDigits)

Converts a number to exponential notation with a specified number of decimal places.

const result = tb.transform(tb.pipe(tb.number(), tb.toExponential(2)), 42000);
// result: "4.20e+4"
toLocaleNumberString(locales, options)

Formats a number according to locale-specific conventions.

const result = tb.transform(
  tb.pipe(
    tb.number(), 
    tb.toLocaleNumberString('en-US', { style: 'currency', currency: 'USD' })
  ), 
  42.5
);
// result: "$42.50"

Number Constraints

toMinValue(min)

Ensures a number is at least a specified minimum value.

const result = tb.transform(tb.pipe(tb.number(), tb.toMinValue(10)), 5);
// result: 10
toMaxValue(max)

Ensures a number is at most a specified maximum value.

const result = tb.transform(tb.pipe(tb.number(), tb.toMaxValue(100)), 150);
// result: 100

Combined Number Actions Example

This example demonstrates how to combine multiple number actions to create a price formatter:

import * as tb from 'transformbot';

// Create a reusable transformation for currency values
const priceFormatter = tb.pipe(
  tb.number(),                // Ensure we're working with a number
  tb.toMinValue(0),           // Ensure price is not negative
  tb.toFixed(2),              // Format with 2 decimal places
  tb.custom(price => `$${price}`) // Add currency symbol
);

// Example usage:
const dirtyPrice = "24.9999";
const formattedPrice = tb.transform(priceFormatter, dirtyPrice);
// Result: "$25.00"

Date Actions

These actions operate on Date objects, providing formatting and constraint options.

Date Formatting

toDateString()

Formats a Date object to a date string in the format "YYYY-MM-DD".

const result = tb.transform(
  tb.pipe(tb.date(), tb.toDateString()), 
  new Date(2023, 5, 15)
);
// result: "2023-06-15"
toTimeString(useSeconds, useMilliseconds)

Formats a Date object to a time string (e.g., "14:30:00").

const result = tb.transform(
  tb.pipe(tb.date(), tb.toTimeString(true, false)), 
  new Date(2023, 5, 15, 14, 30, 45)
);
// result: "14:30:45"
toLocaleDateString(locales, options)

Formats a Date object to a localized date string.

const result = tb.transform(
  tb.pipe(
    tb.date(), 
    tb.toLocaleDateString('en-US', { weekday: 'long', year: 'numeric', month: 'long', day: 'numeric' })
  ), 
  new Date(2023, 5, 15)
);
// result: "Thursday, June 15, 2023"
toLocaleTimeString(locales, options)

Formats a Date object to a localized time string.

const result = tb.transform(
  tb.pipe(
    tb.date(), 
    tb.toLocaleTimeString('en-US', { hour: '2-digit', minute: '2-digit' })
  ), 
  new Date(2023, 5, 15, 14, 30)
);
// result: "02:30 PM"

Date Constraints

toMinValue(minDate)

Ensures a date is not earlier than a specified minimum date.

const minDate = new Date(2023, 0, 1);
const result = tb.transform(
  tb.pipe(tb.date(), tb.toMinValue(minDate)), 
  new Date(2022, 11, 31)
);
// result: minDate (2023-01-01)
toMaxValue(maxDate)

Ensures a date is not later than a specified maximum date.

const maxDate = new Date(2023, 11, 31);
const result = tb.transform(
  tb.pipe(tb.date(), tb.toMaxValue(maxDate)), 
  new Date(2024, 0, 1)
);
// result: maxDate (2023-12-31)

Combined Date Actions Example

This example demonstrates how to combine multiple date actions to create a date validator and formatter:

import * as tb from 'transformbot';

// Create a reusable transformation for a birth date
const birthDateFormatter = tb.pipe(
  tb.date(),                                  // Ensure we're working with a Date
  tb.toMinValue(new Date(1900, 0, 1)),       // Must be after 1900
  tb.toMaxValue(new Date()),                 // Cannot be in the future
  tb.toLocaleDateString('en-US', {           // Format as Month Day, Year
    year: 'numeric',
    month: 'long',
    day: 'numeric'
  }),
  tb.toNull()                                // Convert invalid dates to null
);

// Example usage:
const dirtyDate = "2000-06-15";
const formattedDate = tb.transform(birthDateFormatter, dirtyDate);
// Result: "June 15, 2000"

Normalization Actions

These actions handle edge cases like null values, empty strings, or default values.

Null/Empty Handling

toNull(options)

Converts empty or zero values to null based on their type.

The following values are converted to null by default:

  • Empty arrays: []
  • Date with time 0: new Date(0)
  • Numbers: 0 and 0.0
  • Empty strings: ""

You can customize this behavior with options:

  • skipArray: Don't convert empty arrays to null
  • skipDate: Don't convert Date(0) to null
  • skipNumber: Don't convert 0/0.0 to null
  • skipString: Don't convert empty strings to null
// Default behavior
const result1 = tb.transform(tb.pipe(tb.string(), tb.toNull()), "");
// result1: null

// With options
const result2 = tb.transform(
  tb.pipe(tb.number(), tb.toNull({ skipNumber: true })), 
  0
);
// result2: 0 (not converted to null)
toUndefined(options)

Similar to toNull(), but converts values to undefined instead.

Accepts the same options as toNull().

const result = tb.transform(tb.pipe(tb.string(), tb.toUndefined()), "");
// result: undefined
toDefault(defaultValue)

Replaces null values with a specified default value.

const result = tb.transform(
  tb.pipe(tb.string(), tb.toNull(), tb.toDefault("N/A")), 
  ""
);
// result: "N/A"

Combined Normalization Actions Example

This example demonstrates how to combine normalization actions with other transformations to create a robust data processor:

import * as tb from 'transformbot';

// Create a reusable transformation for a form field
const phoneNumberFormatter = tb.pipe(
  tb.string(),                   // Ensure we're working with a string
  tb.trim(),                     // Remove surrounding whitespace
  tb.toDigits(),                 // Keep only digits
  tb.toNull(),                   // Convert empty strings to null
  tb.toDefault("Not provided")   // Replace null with a default message
);

// Example usage with different inputs
const values = [
  " (123) 456-7890 ",  // Normal phone number with formatting
  "",                   // Empty string
  null                  // Null value
];

const results = values.map(value => tb.transform(phoneNumberFormatter, value));
// Results: ["1234567890", "Not provided", "Not provided"]

Custom Actions

Create your own transformation functions for specialized use cases.

custom(transformFn)

The custom() function allows you to create your own transformation logic by providing a function that takes an input value and returns a transformed output value. This is useful for specialized transformations that aren't covered by the built-in actions.

You can use the custom() function with any type of input and output.

import * as tb from 'transformbot';

// Simple custom transformation for a string
const result1 = tb.transform(
  tb.pipe(
    tb.string(),
    tb.custom(value => value.split('').reverse().join(''))
  ),
  "hello"
);
// result1: "olleh" (reversed string)

// Custom transformation for a number
const result2 = tb.transform(
  tb.pipe(
    tb.number(),
    tb.custom(value => value * value)
  ),
  5
);
// result2: 25 (squared value)

// Custom transformation that changes the type
const result3 = tb.transform(
  tb.pipe(
    tb.number(),
    tb.custom(value => ({ value, squared: value * value }))
  ),
  5
);
// result3: { value: 5, squared: 25 } (object with original and squared values)

Complex Custom Action Example

This example demonstrates how to create and use a more complex custom transformation function:

import * as tb from 'transformbot';

// Create a custom formatter for credit card numbers
const formatCreditCard = tb.custom(value => {
  if (value === null || value === undefined) {
    return null;
  }

  // First ensure we're working with a string
  let cardStr = String(value);

  // Remove all non-digit characters
  cardStr = cardStr.replace(/\D/g, '');

  // Check if it's a valid length for a credit card
  if (cardStr.length < 13 || cardStr.length > 19) {
    return null; // Invalid length
  }

  // Format with spaces every 4 digits
  const formatted = cardStr.match(/.{1,4}/g)?.join(' ');

  // Mask all but the last 4 digits
  const lastFourDigits = cardStr.slice(-4);
  const maskedPart = cardStr.slice(0, -4).replace(/./g, '*');
  const maskedNumber = maskedPart.match(/.{1,4}/g)?.join(' ') + ' ' + lastFourDigits;

  return {
    raw: cardStr,
    formatted: formatted,
    masked: maskedNumber
  };
});

// Use the custom formatter in a transformation pipeline
const creditCardProcessor = tb.pipe(
  tb.string(),
  tb.trim(),
  formatCreditCard,
  tb.toNull()
);

// Example usage
const cardInput = "4111-1111-1111-1111";
const processedCard = tb.transform(creditCardProcessor, cardInput);
// Result: {
//   raw: "4111111111111111",
//   formatted: "4111 1111 1111 1111",
//   masked: "**** **** **** 1111"
// }
